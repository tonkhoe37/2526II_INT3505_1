# OpenAPI (Swagger) - Library API

## Giới thiệu

Thư mục này chứa tài liệu API sử dụng OpenAPI Specification (Swagger) cho hệ thống quản lý thư viện.

## File chính

- `openapi.yaml`: Định nghĩa API

## Cài đặt

Cài Swagger UI CLI:

npm install -g swagger-ui-cli

## Chạy tài liệu API

swagger-ui-cli serve openapi.yaml

## Truy cập

http://localhost:8080

## Test API

- Sử dụng Swagger UI
- Hoặc import file vào Postman
