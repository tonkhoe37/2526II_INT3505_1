# RAML - Library API

## Giới thiệu

Thư mục này chứa tài liệu API sử dụng RAML (RESTful API Modeling Language).

## File chính

- `api.raml`: Định nghĩa API

## Cài đặt

npm install -g raml2html

## Chuyển sang HTML

raml2html api.raml > index.html

## Mở file

Mở file index.html bằng trình duyệt

## Test API

- Chỉ dùng để đọc tài liệu
- Có thể kết hợp với tool khác để test
