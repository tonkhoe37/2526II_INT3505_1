# API Blueprint - Library API

## Giới thiệu

Thư mục này chứa tài liệu API sử dụng API Blueprint (định dạng Markdown).

## File chính

- `api.apib`: Tài liệu API

## Cài đặt

npm install -g aglio

## Chạy tài liệu API

aglio -i api.apib -s

## Truy cập

http://localhost:3000

## Test API

- Xem trực tiếp trên giao diện HTML
- Không hỗ trợ test trực tiếp như Swagger
