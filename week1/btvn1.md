3 API công khai phổ biến :

- Github API là REST API cung cấp khả năng truy vấp và thao tác dữ liệu trên GitHub thông qua HTTP requests cho phép lập trình viên quản lí mã nguồn và cộng tác dự án 1 cách tự động dễ dàng hơn trên nền tảng GitHub

Chức năng :
+Lấy thông tin người dùng
+Quản lý repository
+Quản lý issus và pull requests

Ứng dụng : DevOps tools, dashboard quản lý code, tự động hóa CI/C

- OpenWeather API là API cung cấp dữ liệu thời tiết toàn cầu, bao gồm:
  +thời tiết hiện tại
  +dự báo
  +dữ liệu lịch sử
  +cảnh báo thời tiết.
  API sử dụng REST và trả dữ liệu dạng JSON.

Ứng dụng : Ứng dụng vào dự báo thời tiết , IOT , hệ thống nông nghiệp thông minh

- Google Maps API là tập hợp các API cho phép truy cập dữ liệu bản đồ, vị trí và định tuyến của Google Maps.

Các API phổ biến gồm:
+Maps JavaScript API
+Geocoding API
+Directions API
+Places API
+Distance Matrix API

Chức năng:
+Hiển thị bản đồ
+Tìm địa điểm
+Tính khoảng cách và chỉ đường

Ứng dụng : Ứng dụng gọi xe , giao hàng , du lịch và định vị
